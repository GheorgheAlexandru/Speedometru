# KiCad Speedometer Cluster Project

## Overview
This project involves designing a speedometer cluster using KiCad, an open-source electronic design automation (EDA) software. The speedometer cluster will display speed readings and other relevant data for automotive applications.

## Features
- Digital display for speed readings
- Optional analog gauges for RPM and fuel level
- Inputs for speed sensors and other peripherals
- Customizable PCB layout

## Getting Started
### Prerequisites
- KiCad installation (Version 6.0 or later recommended)
- Basic knowledge of electronics and PCB design

### Installation
1. Clone this repository:
    ```bash
    git clone https://github.com/GheorgheAlexandru/Speedometru.git
    ```
2. Open the KiCad project file (`.kicad_pcb`) in KiCad.
3. Review the schematic and layout, make any required adjustments based on your needs.

## Usage
Once you have downloaded and opened the project, you can:
- Modify the schematic to add or change components.
- Update the PCB layout as needed for manufacturing.
- Export Gerber files for PCB production.
